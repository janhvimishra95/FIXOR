# 🚀 Fixor – Home Services Platform

✨ A modern full-stack web application that connects users with trusted home service professionals.

---

## 🌟 Features

- 🏠 Home Services (Plumber, Electrician, Cleaner)
- 💄 Services for Her (Facial, Hair Spa)
- 🧔 Services for Him (Haircut, Grooming)
- 🧒 Services for Child (Babysitting, Kids Haircut)
- 🔍 Search functionality
- 📦 Booking system with popup
- 🔐 Login UI
- 💾 MongoDB integration (stores bookings)
- 🎨 Attractive UI with animations

---

## 🛠️ Tech Stack

- **Frontend:** React.js, CSS  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB
